# gen-db-dict

#### 介绍
数据库字典生成工具，目前支持生成markdown文档

#### 软件架构
php
composer

### 生成预览
> 将markdown贴到语雀后的效果
![](https://s3.bmp.ovh/imgs/2022/05/21/2ffb7f18d45567c7.png)

### 使用
`git clone https://github.com/dxmq/gen-db-dict.git`

`cd gen-db-dict`

`php index.php`

就会在当前目录生成数据字典markdown文档
